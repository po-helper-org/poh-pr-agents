Create the PR-Agent GitHub App

GitHub Apps can only be created in the browser (no API). Fill the two fields, click the button — GitHub opens a pre-filled creation page. Press Create GitHub App there. It auto-generates the App ID, private key and webhook secret.

After you click Create

GitHub redirects to <base>/?code=… (the page may not load — fine). Copy the code from the address bar and run:

bash finish-registration.sh <code>

That writes dokploy.env with GITHUB_APP_ID, GITHUB_WEBHOOK_SECRET and GITHUB_PRIVATE_KEY_B64 ready to paste into Dokploy. Add your OPENAI_KEY and deploy.
